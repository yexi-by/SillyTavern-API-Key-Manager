# 变更记录

## 0.3.11

- 优化侧边竖条与悬浮球之间的形态过渡，使用同一视觉层完成圆角、尺寸、阴影和颜色渐变变化。
- 为悬浮球状态文字和连接状态点增加淡入淡出，降低竖条和圆球切换时的视觉割裂感。
- 修复贴边唤醒时定位受动画宽度影响，导致圆球可能越出屏幕的问题。

## 0.3.9

- 贴边收缩态改为系统悬浮窗风格的屏幕侧边竖条，不再以半个圆形悬浮球占用界面空间。
- 支持从侧边竖条向屏幕内横向滑动唤醒完整悬浮控制台，竖向滑动仍可调整侧边入口位置。

## 0.3.8

- 悬浮球拖到左右屏幕边缘后会自动停靠并收缩成侧边入口。
- 为悬浮控制台展开、收起和侧边收缩增加平滑过渡动画，并尊重系统减少动态效果设置。

## 0.3.7

- 悬浮控制台改为根据悬浮球位置自适应展开，避免靠近屏幕边缘时覆盖悬浮球或溢出视口。

## 0.3.6

- 过滤服务切换和密钥轮换时可能残留的“连接与密钥已由 API Key 管家接管”提示。

## 0.3.5

- 原 API 连接页被接管时不再额外弹出颜色提示，只保留页面内接管说明。
- 悬浮球服务简称改为单字符显示；英文服务名自动使用大写首字母。

## 0.3.4

- 修复原生连接字段被清空后可能无法自动恢复的问题。
- 修复启用开关被点击时重复触发启用或停用逻辑的问题。
- 修复当前模型测试期间切换模型后测试结果显示错误模型名的问题。
- 限制附加请求体覆盖 `model`、`messages`、`prompt`、`stream` 等由控制台托管的字段。
- 为模型获取和当前模型测试增加超时处理，避免服务商无响应时界面长期停留在加载状态。
- 补齐发布所需的许可证、发布说明和独立仓库忽略规则。
